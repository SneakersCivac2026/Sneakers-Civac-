# Sneakers Civac 👟

Tienda web profesional para la venta de tenis de calidad.

## 📋 Características

✅ Diseño moderno y responsivo
✅ Carga dinámica de productos desde JSON
✅ Formulario de contacto funcional
✅ Navegación fluida y accesible
✅ Optimizado para dispositivos móviles
✅ Fácil de mantener y extender

## 📁 Estructura de archivos

```
Sneakers-Civac_/
├── index.html          # Página principal
├── styles.css          # Estilos y diseño responsivo
├── script.js           # Lógica interactiva
├── productos.json      # Base de datos de productos
└── README.md           # Este archivo
```

## 🚀 Cómo usar

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/SneakersCivac2026/Sneakers-Civac-.git
   ```

2. **Abre el archivo `index.html` en tu navegador**

3. **Personaliza los productos:**
   - Edita `productos.json` con tus propios productos
   - Cambia las imágenes, nombres y precios

4. **Personaliza los estilos:**
   - Modifica `styles.css` para ajustar colores y diseño

## 📝 Formato de productos.json

```json
[
  {
    "nombre": "Nombre del tenis",
    "precio": 899,
    "moneda": "MXN",
    "img": "URL de la imagen"
  }
]
```
